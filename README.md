# Diagram Templates
template diagram มาตรฐานสำหรับงาน network / system / infra

## โครงสร้าง folder
- `network/templates/` — Network diagram (.drawio)
- `system/` — System architecture diagram
- `infra/` — Infrastructure / rack diagram
- `prompts/` — Prompt สำเร็จรูปสำหรับ Claude

## naming convention
ตั้งชื่อไฟล์แบบนี้
network-[ชื่อลูกค้า]-v[เลขversion].drawio
เช่น network-abc-company-v1.drawio

## วิธีใช้
1. เปิดไฟล์ .drawio ด้วย draw.io desktop หรือ draw.io online
2. แก้ไขตามต้องการ
3. export เป็น .pdf ก่อนส่งลูกค้า
