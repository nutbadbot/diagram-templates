# 📸 Examples

Folder นี้เก็บ **ตัวอย่างการใช้งานจริง** (before/after, case studies)

## โครงสร้าง

```
examples/
├── before-after/    ← ตัวอย่าง diagram ก่อน-หลัง Claude แก้
└── case-studies/    ← case ที่ใช้งานจริง (ลบข้อมูลลูกค้าแล้ว)
```

## 🚧 ยังว่างอยู่

หลังทีมทดลองใช้จริงแล้ว ค่อยเอาเคสมาใส่ที่นี่ (mask ข้อมูลลูกค้าออกก่อน)
